Question - 1: What are the different ways to select an element in the DOM?

There are five ways we can select an element in the DOM which are by id(document.getElementById('id')), by class(document.getElementsByClassName('class')), by tag(document.getElementsByTagName('tag')), by css selector(document.querySelector('selector')), and by name(document.getElementsByName('name')).

Question - 2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Returns or sets the HTML content, including tags. innerText: Returns or sets the visible text, excluding hidden elements and ignoring styling. textContent: Returns or sets all text, including hidden elements, without HTML tags.

Question - 3: What is the event delegation in the DOM?

Event delegation is a technique where we can attach a single event listener to a parent element to manage events for all its child elements. It leverages event bubbling, improving performance and reducing memory usage, especially for dynamic content.

Question - 4: What is the event bubbling in the DOM?

Event bubbling is a DOM event propagation mechanism where an event triggered on a child element propagates up through its ancestors in the hierarchy. This allows parent elements to handle events originating from their children.

Question - 5: How do you create, add, and remove elements using JavaScript?

To create element we can use document.createElement('tag name'). To add element we can use parentElement.appendChild(newElement). To remove element we can use parentElement.removeChild(childElement).
